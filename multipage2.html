Multipage Tour Landing Page

Here we are!

Note that on the second page, we'll need to include hopscotch.js as well as the tour javascript file (in this case, multipage-demo.js). On page 2, you can use hopscotch.getState() to determine if you should start the tour automatically.


        if (hopscotch.getState() === "hello-hopscotch:1") {
          hopscotch.startTour(tour);
        }

hopscotch.getState() returns a string of the form "tourId:tourStepNumber". Use it to make sure that the user is in the middle of the tour before starting the tour automatically.

In this example, we used the onNext callback to navigate to the second page. Another option is to disable the next button and have your Hopscotch step point to a link on the page. Such a tour would look something like the following:


        var tour = {
          id: "hello-hopscotch",
          steps: [
            {
              title: "Multipage Example",
              content: "We're going to the next page now...",
              target: "#page2-link",
              placement: "bottom",
              multipage: true,
              showNextButton: false
            },
            {
              title: "Here we are!",
              content: "Safe and sound!",
              target: "explanation",
              placement: "bottom",
              xOffset: "center",
              arrowOffset: "center"
            }
          ]
        };
